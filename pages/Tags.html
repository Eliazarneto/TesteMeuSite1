Tags

  • hr - linha horizontal

  • h1 - ate h6 - criação de titulos

  • p - paragrafo
  • br - quebra de linha
  • img - adicionar imagem
  • link:favicon - adicionar icone na pagina
  • a - cria links/botoes
  • EM - ITALICO com semantica
  • strong - negrito com semantica
  • mark - marca texto - mark style="background-color: aliceblue;)
  • Del - riscar texto
  • ins - sublinhar texto
  • Sup - números elevados
  • sub - números abaixo das letras
  • code - permite digitação do código
  • q - citação com semantica
  • pre - digita codigo com formatação certa
  • abbr - citação dentro de uma palavra/frase
  • ol - lista organizada
  • ul - lista sem organização
  • li - criação de topicos de lista
  • ../ - topico para procura de arquivos em pastas
  • audio - tag para reprodução de audio
  • Ao criar uma tag de audio, você pode criar um source, para que o navegador reproduza o conteudo
  • video - tag para adicionar video

Css

  • style - usado para alterar o visual
  • Dentro do style, voce pode alterar a cor dos itens criando a tag especifica, como por exemplo h2{} irá mudar todos os h2 do texto
  • CTRL CLICK - acaba criando uma pasta ou arquivo com o nome que já está lá
  • Se voce colocar o style dentro da pagina html, a modificação irá ficar para aquela pagina, exemplo, colocar o texto h1 em negrito, irá ficar apenas lá, porém, alterando dentro da css, todas as páginas linkadas receberão o update, assim como se apenas uma linha ou parágrafo precisar de alteração.
  • Background-image - possibilita criação de gradientes e afins
  • ID - cria uma categoria para que cada tag seja editada por grupo. no css, so colocar # e a id
  • class - tag para separar os elementos.
  • Span - tag que permite adicionar class dentro do texto
  • border - permite a altração nas bordas
  • margin - altera a margem - espaço externo
  • padding - altera a borda, espaço interno
  • outline - tracejado dentro da margem
  • nav, section, article, aside - Todas tags semanticas para criar grouping tags
  • Box-shadow - 1px 3x e a cor - cria sombra em caixa
