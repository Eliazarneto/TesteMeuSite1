tipografia

tipografia é o estudo da escrita, como escrever e representar emoções com as letras. Assim como com as cores.

Anatomia dos tipos

Anatomia dos tipos é a forma como cada letra é escrita. Cada letra ocupa um espaço diferente, você precisa ver como ela se encaixa no seu site.

  • Altura das Maiúsculas - A altura normalmente é um pouco menor que a soma da ascendente com a mediana.
  • Ascendente - Parte das letras maiúsculas que passam da linha mediana - altura x.
  • Descendente - Parte de baixo que passa da linha de base.
  • Altura x - Define o tamanho das letras minúsculas, baseada na letra x.
  • Corpo - É o tamanho total de uma letra, o valor que podemos mudar no style.
  • Margin fica da borda do conteudo para fora.
    O padding fica da borda para dentro do conteudo.
    Border é a configuração da própria borda.
  • Margin auto faz com que seja jogado para o centro
