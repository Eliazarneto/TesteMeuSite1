Começando primeiramente com as TAGS

- HR significa linha horizontal
- h1 significa titulo - dentro do h1, podem existir h2, h3, h4 ate o 6, assim como podem existir o h2, h3, h4, dentro do h2. para mostrar que aqueles topicos estão dentro do titulo principal, eles são divididos por níveis
- p é paragrafo
- "tudo é chamado de marca ou tag derivado do html"
- br - break role, quebra de linha
- & lt; seguido por & gt; é o comando que possibilita a digitação dos seguintes caracteres < >, ja que utilizando os mesmos diretamente, eles irão criar uma tag.
- !<-- (é a tag para criação de comentario)
- &#x - é o comando para criação de emoji
- img - é o comando para adicionar imagem, voce póde adicionar imagem de uma pasta, subpasta, ou link url de qualquer lugar.
- link:favicon - é o comando para adicionar o icone da pagina, ele deve ser adicionado na parte HEAD do programa, normalmente acima da linha title

- header - indica que tudo que estiver dentro dele é do cabeçalho
- main - tag que cria a parte do meio da página
- footer - tag semantica que separa o rodapé
- nav - menu de navegação, ancoras, links, que voce clica para ir ao outro lugar. dentro dele se usa a tag a.
- a - tag para criar botoes que levam a outro lugar.
- section - seção, para dividir o meio da página. a main
- strong - comando que deixa o texto dentro dele em negrito. voce inicia com a abertura da tag e finaliza com o fechamento.
- a tag b, também serve para deitar em negrito. porém, ela é uma tag sem semantica.
- EM - é a tag para a marcação em negrito. o comando com semantica
- I - É atag para a criação de negrito sem semantica.
----------- AS TAGS B E I NÃO SÃO MUITO UTILIZADAS --------
- Mark é uma tag que praticamente gera um marca texto no texto selecionado. (Para mudar a cor voce usa css dentro da propria tag, apertando espaço e criando o parametro style.ex mark style="background-color: aliceblue;) ---- Ao criar a categoria style, dentro do head, para a tag mark, voce pode escolher a cor para todas as marcações do texto
- Del - é uma tag para riscar o texto, deletado
- Ins - é uma tag para sublinhar o texto, inserido (Pode ser utilizada a tag U, mas ela não é semantica)
- Sup - tag para colocar numeros elevados como x20
- sub - tag para colocar numeros abaixo como h2O
- code - tag que serve para marcar um codigo
- q - é uma tag que coloca as aspas em uma citação, onde voce da sentido a citação diferente de fazer apenas o uso das aspas "" 
- pre - tag para colocar os codigos exatamente como sao digitados
dentro da tag pre, da pra criar uma formatação diferente, onde o texto vai ser guiado pelas quebras de linha e tudo 
(porém não é recomendado, ja que acaba virando bagunça)

- blockquote - tag que cria uma citação, e dentro dela voce pode adicionar "cite" e colocar o link, para que o site saiba que voce esta citando tal livro;
- abbr - tag para abreviações. o texto terá uma abreviação que voce verá ao colocar o mouse em cima;

OBSERVAÇÕES E TAGS IMPORTANTES

- a criação da tag style dentro do head, permite a alteração em cores do nosso script, atraves das tags especificas, seja do body, header, mark, etc; - OL - Para criar uma lista organizada. dentro dele vc coloca o li para cada topico.
Dentro do OL você pode adicionar o Type
- UL - tag para listas desordenadas, apenas com as bolinhas.
- também é possivel adicionar uma lista dentro de outra lista.
- Dl é uma lista de definição, dentro dela voce cria a primeira variavel com a tag dt, e depois a segunda, com a tag dd, que será nome seguido de significado.
- a - é a tag para criação de hyperlinks, atraves do envelopamento, ctrl e espaço permite escolher uma segunda página
- CTRL SHIFT P - COMANDO PARA ENVELOPAR - target - tag para que o link seja aberto em uma pagina em branco, exemplo de uso target = "_blank" seguido por
- rel - seguido de "external", significando que é um link externo (seguir EX010)
- Para criação de outras páginas, voce apenas precisa deixa-las na mesma pasta - rel next proxima, rel prev previous - ajuda o sistema de pesquisa a encontrar o seu site
- rel nofollow - serve para que o mecanismo de busca não faça o index entre o seu site e do terceiro. - target _blank - pagina em branco - target _self - propria pagina
- ../ - permite que voce vá para duas pasta antes, assim como ./ sobe uma pasta.
